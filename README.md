# Prodigy_ds_02
1. Create your project folder and open it in VS Code
Create a folder somewhere on your PC, for example:
text
C:\Users\YourName\titanic_eda
Open VS Code.
From the menu: File → Open Folder and select titanic_eda.
Optional: in the folder, create a output subfolder for results later.

2. Install Python and the required packages
Make sure Python is installed (e.g., from python.org). You should have python and pip available.
Open the integrated terminal in VS Code:
Press Ctrl + Shift + P
Run View: Terminal (or just press Ctrl + Shift + ß).
In the terminal, install the libraries:
pip install pandas seaborn matplotlib

3.In VS Code's Explorer (left sidebar), click New File.
Name it:
text
titanic_eda_report.py    then paste the code

4. Run the script in VS Code
Open titanic_eda_report.py in VS Code.
In the integrated terminal (same as before), make sure your current directory is the project folder:
bash
cd titanic_eda
Run the script:
bash
python titanic_eda_report.py
Wait for it to finish. If it completes successfully, you'll see:
text
Done. Single PDF created: titanic_eda_report.pdf
